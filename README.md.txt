# Test Technique Data Scientist - Geoptis

Ce dépôt contient la résolution du test technique pour le poste de Data Scientist chez Geoptis. L'étude porte sur l'analyse socio-économique et géographique du marché locatif en France métropolitaine.

## Présentation du projet
L'objectif est d'explorer un jeu de données communales pour comprendre la structure des loyers moyens et leur corrélation avec la précarité. Le projet suit un workflow complet, du nettoyage des données brutes à la cartographie régionale.

## Méthodologie
- **Langage :** Python (Numpy, Geopandas, Seaborn)
- **Traitement :** Nettoyage des données aberrantes.
- **Visualisation :** Analyses statistiques (Distribution) et Cartographie (Nouvelle-Aquitaine).

## Structure du projet
- `Souhayal_Elkaroumi_Notebook_Test_Technique_Geoptis.ipynb` : Le notebook principal contenant l'intégralité de l'analyse, du code et des interprétations.
- `data/` : Dossier contenant le fichier source (RDS), le dataset exporté après nettoyage (CSV) ainsi que la data brut en format GeoJSON.
- `README.md` : Présentation et méthodologie.
